# Árvore Proposta Para Seletivos no IndicaJur

## 1. Seletivos Para Candidatos

Objetivo: permitir que o candidato encontre oportunidades compatíveis, entenda o suficiente para decidir e se inscreva sem ruído.

### Menu do candidato
- Meu Perfil
- Editar Perfil
- Ver Seletivos
- Sair

### Ver Seletivos
- Lista de Seletivos abertos.
- Filtros simples: instituição, área, estado, regime e prazo.
- Card do Seletivo com:
  - órgão/instituição;
  - local;
  - área;
  - regime;
  - prazo de inscrição;
  - etapas previstas;
  - status da inscrição do candidato.
- Botão principal:
  - `Ver detalhes`, antes da decisão.
  - `Inscrever-se`, apenas se o candidato estiver logado.
  - `Inscrito`, quando já inscrito.

### Detalhe do Seletivo
- Descrição objetiva da função.
- Requisitos desejáveis.
- Etapas e prazos.
- Aviso de compartilhamento de perfil.
- Confirmação expressa para inscrição.

### Minhas Inscrições
Etapa futura recomendada.
- Seletivos inscritos.
- Status da inscrição.
- Próxima etapa prevista.
- Mensagens ou avisos do recrutador.

## 2. Seletivos Para Recrutadores

Objetivo: dar ao recrutador uma ferramenta de gestão confiável, sem transformar o processo em burocracia inútil.

### Menu do recrutador
- Perfil
- Editar Perfil
- Seletivos
- Banco de Candidatos
- Sair

### Perfil
- Dados institucionais.
- Atalhos:
  - Novo Seletivo.
  - Meus Seletivos.

### Novo Seletivo
- Dados da oportunidade:
  - órgão;
  - cargo/função;
  - área ou áreas;
  - local;
  - regime;
  - remuneração, se divulgável;
  - prazo final de inscrição.
- Etapas:
  - triagem curricular;
  - entrevista estruturada;
  - prova/teste de escrita;
  - análise de documentos;
  - período de experiência;
  - outra etapa.
- Para cada etapa selecionada:
  - data final prevista;
  - orientação curta sobre como será conduzida.

### Meus Seletivos
- Lista de Seletivos do recrutador.
- Ações por Seletivo:
  - editar;
  - ver inscritos;
  - acompanhar favoritos;
  - registrar anotações;
  - encerrar;
  - duplicar como modelo.

### Inscritos
- Lista de candidatos inscritos naquele Seletivo.
- Filtros internos:
  - formação;
  - experiência;
  - área;
  - OAB;
  - disponibilidade;
  - sistemas;
  - DISC;
  - selos.
- Ações:
  - favorito;
  - anotação;
  - status interno;
  - pré-selecionado;
  - entrevistado;
  - recomendado;
  - descartado.

## 3. Regra Central de Produto

O mesmo Seletivo precisa ter duas interfaces:

- Para o candidato: descoberta, clareza e inscrição.
- Para o recrutador: gestão, comparação e decisão.

O candidato não deve enxergar ferramentas internas de triagem.  
O recrutador não deve depender da tela pública para administrar o processo.
